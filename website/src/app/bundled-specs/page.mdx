# Bundled specs

The npm package ships **`providers/`**—optional on-disk API descriptions so cold start avoids downloading multi‑MB specs. **`CLAWQL_PROVIDER`** selects a **single vendor**, a **merged preset**, or interacts with [Spec configuration](/spec-configuration) precedence. {{ className: 'lead' }}

## Presets and vendors

| `CLAWQL_PROVIDER`                                                                                                   | Notes                                                                                                                   |
| ------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| `google`                                                                                                            | Merged **Google Cloud** APIs from the on-disk manifest (see repo `providers/google/`; manifest filename is historical)  |
| `default-multi-provider`                                                                                            | Same as **no spec env**: Google Cloud (bundled) + Cloudflare + GitHub + Slack + Paperless + Stirling + Tika + Gotenberg |
| `all-providers`                                                                                                     | Google Cloud (bundled) **plus** every other bundled vendor (Jira, Bitbucket, Sentry, n8n, …)                            |
| `atlassian`                                                                                                         | Jira + Bitbucket merged                                                                                                 |
| `cloudflare`, `github`, `slack`, `paperless`, `stirling`, `tika`, `gotenberg`, `sentry`, `n8n`, `jira`, `bitbucket` | Single bundled OpenAPI/Discovery per vendor                                                                             |

Aliases: **`jira`** / **`bitbucket`** map to their standalone bundles; **`atlassian`** merges both. Deprecated merged id: **`google-top50`** → **`google`**.

## Offline and refresh

Bundled files are read **from disk first**. If a file is missing, the registry may fetch a **fallback URL** unless **`CLAWQL_BUNDLED_OFFLINE=1`**.

Maintainers refresh specs with **`npm run fetch-provider-specs`** (see [`providers/README.md`](https://github.com/danielsmithdevelopment/ClawQL/blob/main/providers/README.md)). Optional GraphQL artifacts: **`npm run pregenerate-graphql`** (Bun recommended).

## Default first run

With **no** spec-related environment variables, ClawQL loads **`default-multi-provider`** so cross-vendor **`search`** works immediately after install.
