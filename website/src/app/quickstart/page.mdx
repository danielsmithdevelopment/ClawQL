# Quickstart

You can be up and running in a few minutes with **`npx`**—no global install required. Run **`clawql-mcp`**; **`execute`** uses in-process OpenAPI→GraphQL for single-spec APIs. {{ className: 'lead' }}

<Note>
  The published package is **large on disk** (~90+ MB) because it ships bundled specs under **`providers/`**. That is intentional for offline discovery.
</Note>

## 1. Start the MCP server

```bash
npx -p clawql-mcp clawql-mcp
```

For every bundled vendor at once:

```bash
CLAWQL_PROVIDER=all-providers npx -p clawql-mcp clawql-mcp
```

## 2. Connect your client

Add an MCP server entry that runs **`clawql-mcp`** on stdio (see [MCP clients](/mcp-clients)). No separate GraphQL process or **`GRAPHQL_URL`** is required.

## 3. Use your own API spec

**Local file**

```bash
CLAWQL_SPEC_PATH=./openapi.yaml npx -p clawql-mcp clawql-mcp
```

**Remote OpenAPI**

```bash
CLAWQL_SPEC_URL=https://example.com/openapi.json npx -p clawql-mcp clawql-mcp
```

**Google Discovery**

```bash
CLAWQL_DISCOVERY_URL="https://compute.googleapis.com/\$discovery/rest?version=v1" npx -p clawql-mcp clawql-mcp
```

## 4. HTTP transport (optional)

For **`clawql-mcp-http`**, MCP is at **`/mcp`** and GraphQL at **`/graphql`** on the same **`PORT`** (see [Install](/install)).
