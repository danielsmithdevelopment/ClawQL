# ClawQL documentation

**ClawQL** registers **core** tools **`search`** and **`execute`**, **`audit`** and **`cache`** (always on), plus **`sandbox_exec`** (remote code in a Cloudflare Sandbox via bridge), **`ingest_external_knowledge`** (bulk Markdown + optional URL fetch), and vault **`memory_ingest`** / **`memory_recall`** by default (set **`CLAWQL_ENABLE_MEMORY=0`** to opt out of vault memory tools). **`execute`** covers **OpenAPI 3**, **Swagger 2**, and **Google Discovery** via an internal **OpenAPIŌåÆGraphQL** projection where applicable, and optional **native GraphQL** / **gRPC** upstreams when **`CLAWQL_GRAPHQL_SOURCES`** / **`CLAWQL_GRPC_SOURCES`** are set ([ADR 0002](https://github.com/danielsmithdevelopment/ClawQL/blob/main/docs/adr/0002-multi-protocol-supergraph.md)). The same tool set is available over **stdio**, **Streamable HTTP**, and (when enabled) **MCP over gRPC** via **[`mcp-grpc-transport`](https://www.npmjs.com/package/mcp-grpc-transport)**ŌĆösee [Deployment](/deployment) and [Kubernetes](/kubernetes). [Full tool reference (repo)](https://github.com/danielsmithdevelopment/ClawQL/blob/main/docs/mcp-tools.md). {{ className: 'lead' }}

**`audit`** and **`cache`** (**ClawQL Core**, no opt-out) are **in-process** ŌĆö Core audit ring buffer vs LRU scratch ŌĆö not the vault ŌĆö **[enterprise-mcp-tools.md](https://github.com/danielsmithdevelopment/ClawQL/blob/main/docs/enterprise-mcp-tools.md)** ([#89](https://github.com/danielsmithdevelopment/ClawQL/issues/89)), [Session cache](/cache), [cache-tool.md](https://github.com/danielsmithdevelopment/ClawQL/blob/main/docs/cache-tool.md). Optional **`schedule`** (**`CLAWQL_ENABLE_SCHEDULE`**) adds persisted jobs and synthetic checks ŌĆö [Schedule synthetic checks](/schedule), [schedule-synthetic-checks.md](https://github.com/danielsmithdevelopment/ClawQL/blob/main/docs/schedule-synthetic-checks.md) ([#76](https://github.com/danielsmithdevelopment/ClawQL/issues/76)). Optional **`notify`** (**`CLAWQL_ENABLE_NOTIFY`**) posts to Slack ŌĆö [Slack notify](/notify), [notify-tool.md](https://github.com/danielsmithdevelopment/ClawQL/blob/main/docs/notify-tool.md) ([#77](https://github.com/danielsmithdevelopment/ClawQL/issues/77)). Optional **`knowledge_search_onyx`** (**`CLAWQL_ENABLE_ONYX`**) queries **[Onyx](https://www.onyx.app/)** document search ŌĆö [Onyx knowledge search](/onyx-knowledge), [onyx-knowledge-tool.md](https://github.com/danielsmithdevelopment/ClawQL/blob/main/docs/onyx-knowledge-tool.md) ([#118](https://github.com/danielsmithdevelopment/ClawQL/issues/118)). Optional **`ouroboros_*`** tools (**`CLAWQL_ENABLE_OUROBOROS`**) embed the **[Ouroboros library](/ouroboros)** evolutionary loop, with durable Postgres lineage via **`CLAWQL_OUROBOROS_DATABASE_URL`** (or split **`CLAWQL_OUROBOROS_DB_*`**) ŌĆö [clawql-ouroboros.md](https://github.com/danielsmithdevelopment/ClawQL/blob/main/docs/clawql-ouroboros.md) ([#141](https://github.com/danielsmithdevelopment/ClawQL/issues/141), [#142](https://github.com/danielsmithdevelopment/ClawQL/issues/142)).

## Getting started {{ anchor: false }}

Install **`clawql-mcp`**, run it on stdio, then point your MCP client (e.g. Cursor or Claude Desktop) at the server. Optionally set **`CLAWQL_SPEC_PATH`**, **`CLAWQL_SPEC_URL`**, or **`CLAWQL_DISCOVERY_URL`**; if you set nothing, ClawQL loads a bundled **multi-provider** merge (see [spec configuration](/spec-configuration) ŌĆö default **`all-providers`** includes Google Cloud, Cloudflare, GitHub, Slack, document APIs, Onyx, and more). {{ className: 'lead' }}
